How To Use

  1. Select your starting location from the dropdown.
  2. Enter the surveillance event address.
  3. Tap Open In Google Maps to launch directions, or open Google Maps manually.
  4. Copy the one-way driving distance and time into this page.
  5. Choose One Way or Round Trip.
  6. Tap Calculate Billing.
  7. Enter the returned values in TrackOps Mobile.
Install App Icon How-To Open this page in Chrome, tap the three-dot menu, then tap Add To Home Screen or Install App. If the icon changes, remove the old home-screen shortcut and add it again after the update finishes.
