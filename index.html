Toolbox

Travel calculator, hotel finder, and field tools. Install once, then open from your home screen.

Scan to install

Print this page or save the QR for posters and onboarding. The link should stay the same for company-wide rollout.

Open Toolbox in browser

Follow these steps

    Android (Chrome)
    1. Scan the QR or open this page in Chrome.
    2. Tap Install app above, or the menu → Install app / Add to Home screen.
    3. Open Toolbox from your home screen.
    iPhone (Safari)
    1. Scan the QR or open this link in Safari (not Chrome).
    2. Tap Share → Add to Home Screen → Add.
    3. Open Toolbox from your home screen.

    After installing, allow Notifications and Location when prompted if you use Hotel Finder.
